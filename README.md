# Rainforest Kongra Ecosystem

A Fabric mod for Minecraft 1.20.1 that adds a deadly rainforest ecosystem:

- **Rain Danger**: Stay out in the rain in hot/jungle biomes for too long and you'll begin to lose health.
- **Rainforest Animals**: Rainforest Jaguars (they will hunt you) and colorful Toucans roam the world.
- **The Big Boss – KONGRA**: A 200-HP monster with the body of a gorilla and the head of a king cobra. Without protection, you don't stand a chance.
- **KONGRA Armor**: Crafted from the hardest-to-find rainforest treasures (Rainforest Jade and Cobra Scale). Wearing the FULL set protects you from both the endless rain and KONGRA's devastating extra attacks.

Rainforest Jade and Cobra Scale, all custom items, spawn eggs, and armor are available in the "Rainforest Kongra" creative tab.

## How to build the mod for FREE using GitHub (no Java install needed!)

You do **not** need to install Java or any developer tools. GitHub will build the `.jar` file for you in the cloud, for free.

### Step-by-step

1. **Create a GitHub account** at https://github.com (free).
2. **Create a new repository**: click the **+** in the top-right → **New repository**. Give it any name → click **Create repository**.
3. On the new empty repo page, click the link **"uploading an existing file"**.
4. **Extract the ZIP** you downloaded to a folder on your computer.
5. **⚠️ macOS users — SHOW HIDDEN FILES FIRST!** In Finder, open the extracted folder and press **Cmd + Shift + .** (period). This reveals hidden files. The **`.github`** folder is invisible by default. If you do not show it and upload it, the automatic build will **never run**.
6. **Open the extracted folder** and select **ALL files and folders from INSIDE it** — including the hidden **`.github`** folder. Do **NOT** drag the outer folder itself; drag its **contents**.
7. **Drag all those files/folders** into the GitHub upload area.
8. Scroll down and click **Commit changes**.
9. Click the **Actions** tab at the top of your repo. You'll see a build running.
10. Wait about **2 minutes** for it to finish (green checkmark).
11. Click the finished build run → scroll to **Artifacts** → download **mod-jar**.
12. Unzip the downloaded artifact to get the `.jar` file.
13. Copy the `.jar` into your `.minecraft/mods/` folder (create the `mods` folder if it doesn't exist). Make sure you have **Fabric Loader** and the **Fabric API** installed.
14. Launch Minecraft 1.20.1 with the Fabric profile and enjoy!

### Important reminder for macOS
The `.github` folder is **hidden**. Press **Cmd + Shift + .** in Finder before selecting files, or GitHub won't receive the workflow and the build will never start.

## Survival Tips

- Craft the full KONGRA armor set (helmet, chestplate, leggings, boots) before hunting KONGRA.
- Seek shelter during rain in hot biomes — the rain slowly kills the unprotected.
- Rainforest Jaguars are aggressive; keep your distance or fight back.

Enjoy the jungle — and beware KONGRA.
